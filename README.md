# 📊 Task 04 - Sentiment Analysis

## 🎯 Objective
To analyze social media text data and classify sentiments as positive, negative, or neutral.

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Google Colab  

## 📌 Steps Performed
- Loaded the dataset  
- Assigned column names and explored the data  
- Cleaned text data and removed missing values  
- Visualized sentiment distribution  
- Converted text into numerical form using TF-IDF  
- Split data into training and testing sets  
- Built Naive Bayes model  
- Evaluated model using accuracy and confusion matrix  
- Visualized confusion matrix  

## 💡 Key Insights
- Sentiment distribution varies across categories  
- Text preprocessing is important for NLP tasks  
- TF-IDF converts text into numerical features  
- Naive Bayes works well for sentiment classification  
- Accuracy shows model prediction performance  

## 📷 Output

### 📊 Sentiment Distribution
Sentiment distribution visualization.

### 📊 Confusion Matrix
Confusion matrix showing prediction results.

### 📈 Accuracy Output
Accuracy score for sentiment classification model.
